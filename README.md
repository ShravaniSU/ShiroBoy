# 🐾 Shiro — The Golden Heart of Our Lives

A beautiful emotional storytelling website dedicated to our Golden Retriever, **Shiro**.
Built with ❤️ using **React + TypeScript + Vite**.

---

## ✨ About The Project

This website is a cinematic digital scrapbook that captures Shiro’s journey:

* His adorable puppy days 🐶
* Naughty sofa destruction moments 😭
* Office adventures with Ranjeet 🏢
* His emotional bond with Shravani ❤️
* Tough health battles including epilepsy & hip dysplasia 💔
* And the unconditional love that never changed ✨

The website is designed to feel:

* Warm
* Emotional
* Handcrafted
* Visually immersive
* Modern & premium

## 🚀 Getting Started

### 1. Clone The Repository

```bash
git clone <your-repo-url>
```

### 2. Navigate Into Project

```bash
cd shiro-website
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Start Development Server

```bash
npm run dev
```
